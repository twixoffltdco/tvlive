📖 ТУТОРИАЛ

Как смотреть Zabawa Wink плейлисты

1
Открой плеер по кнопке «Запустить плеер» выше
2
Перейди на GitHub проекта: github.com/CrocoUser/zabava-project
3
Найди .m3u файл в репозитории, нажми на него, затем кнопку «Raw» — скопируй URL
4
Вставь Raw-ссылку в поле плеера и нажми «Загрузить» — каналы появятся автоматически
5
Выбери группу из выпадающего списка и кликни на канал для просмотра
⚠️ Только для РФ: Сервис Zabawa Wink предоставляет каналы для российских пользователей. Используй VPN при необходимости.
