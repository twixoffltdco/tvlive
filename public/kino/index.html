🔒 Политика конфиденциальности

Последнее обновление: 15 января 2025 года

Мы в OINK уважаем вашу конфиденциальность. Эта политика объясняет, какие данные мы собираем, как их используем и защищаем.

1. Какие данные мы собираем

1.1. Cookies и локальное хранилище (localStorage):

  • oink_consent — ваше согласие на использование cookies (true/false)
  • oink_history — история просмотренных фильмов (ID фильмов)
  • oink_genres — предпочтения по жанрам на основе просмотров

1.2. Геолокация (IP-адрес): Мы определяем ваш регион через сервис ipapi.co для отображения флага в шапке сайта. IP-адрес не сохраняется на наших серверах.

1.3. Данные не собираются: OINK НЕ собирает ФИО, адреса, телефоны, паспортные данные, банковскую информацию.

2. Как мы используем ваши данные

  • Рекомендации фильмов: На основе истории просмотров и жанров мы подбираем персональные рекомендации в блоке "Рекомендации для вас".
  • Улучшение сервиса: Анонимная статистика просмотров помогает нам понять, какие фильмы популярны.
  • Отображение региона: Показываем ваш регион в шапке сайта.

3. Cookies и localStorage

Cookies — это небольшие текстовые файлы, которые сохраняются на вашем устройстве. В OINK используются только технически необходимые cookies для работы рекомендаций.

Вы можете контролировать cookies:

  • При первом посещении сайта появляется баннер с запросом согласия
  • Вы можете отклонить cookies — в этом случае персональные рекомендации работать не будут
  • В любой момент можно очистить cookies через настройки браузера

Как очистить cookies в браузере:

  • Chrome: Настройки → Конфиденциальность → Очистить историю → Cookies
  • Firefox: Настройки → Приватность → Удалить cookies
  • Safari: Настройки → Конфиденциальность → Управление данными сайтов

4. Передача данных третьим лицам

OINK НЕ передает ваши данные третьим лицам. Исключения:

  • Плееры Kinobox и Kinobd: При просмотре фильма ваш IP-адрес может передаваться плеерам для загрузки видео. Это необходимо для работы сервиса.
  • API Кинопоиска: Запросы к API содержат только ID фильмов, без персональных данных.
  • ipapi.co: Сервис определения региона получает ваш IP-адрес, но не сохраняет его.

Мы не продаем и не обмениваем ваши данные с рекламными сетями.

5. Хранение и защита данных

5.1. Все данные (история просмотров, жанры) хранятся локально на вашем устройстве в localStorage. Мы не имеем к ним доступа со стороны сервера.

5.2. Сайт работает по протоколу HTTPS, что обеспечивает шифрование данных при передаче между вашим браузером и нашим сервером.

5.3. В случае утечки данных на стороне пользователя (например, взлом устройства) — OINK не несет ответственности.

6. Права пользователей

Вы имеете право:

  • Отказаться от cookies при первом посещении сайта
  • Очистить историю просмотров через настройки браузера
  • Не использовать сайт, если не согласны с политикой конфиденциальности

Так как мы не собираем персональные данные (ФИО, email), запросы на удаление данных не требуются — достаточно очистить cookies в браузере.

7. Дети и несовершеннолетние

OINK не предназначен для лиц младше 18 лет. Мы не собираем данные о возрасте пользователей. Если вы родитель и считаете, что ваш ребенок использовал сайт без вашего согласия — очистите cookies в браузере.

8. Изменения в политике конфиденциальности

Мы можем обновлять эту политику. Актуальная версия всегда доступна по адресу /privacy.html. Продолжая использовать сайт после изменений, вы соглашаетесь с новой версией.

9. Контакты

💡 Совет: Если браузер ТВ не поддерживает сайт, установите приложение "TV Bro" из магазина приложений.

📚 Документация OINK Cinema

Полное руководство по интеграции плееров, использованию API Кинопоиска и работе с платформой OINK.

🎬 Интеграция плеера Kinobox

Рекомендуется Самый стабильный плеер с множеством источников (Alloha, Kodik, VideoCDN, Collaps).

Базовое подключение:

<div id="player"></div>

<script src="https://kinoboxtv.tatnet.app/kinobox.js"></script>
<script>
  kinobox('#player', {
    search: { kinopoisk: 301  // ID фильма с Кинопоиска
    }
  });
</script>

Расширенная конфигурация:

kinobox('#player', {
  search: { kinopoisk: ID },
  players: ['alloha', 'kodik', 'videocdn', 'ashdi', 'collaps'],
  menu: { 
    default: 'menu',
    position: 'top'
  }
});

🎬 Интеграция плеера Kinobd

Альтернатива Простой и быстрый плеер от kinobd.net.

<div data-kinopoisk="301" id="kinobd"></div>
<script src="https://kinobd.net/js/player_.js"></script>

Примечание: ID фильма вставляется в атрибут data-kinopoisk. Плеер автоматически подхватит все доступные источники.

Для получения данных о фильмах, поиска и рекомендаций используется API с ключом:

API_KEY: 960dc4e3-9eae-4072-8f0c-98a5ed6c0d2a

Эндпоинты:

МетодURLОписание

const response = await fetch('https://kinopoiskapiunofficial.tech/api/v2.2/films/301', {

    'Content-Type': 'application/json'
  }

const data = await response.json();
